disassemble

Signature: unsigned int disassemble(unsigned char *bytes, unsigned int max, int offset, char *output)
Description: Disassemble a memory region to the output variable. Returns the bytes disassembled.

Parameters:
  • unsigned char *bytes
  • unsigned int max
  • int offset
  • char *output

Return: unsigned int
