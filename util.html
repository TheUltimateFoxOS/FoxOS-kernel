prepare_memory

Signature: void prepare_memory(stivale2_struct* bootinfo)
Description: Read's the memory map and initialises the PageFrameAllocator also loads a new page table.

Parameters:
  • stivale2_struct* bootinfo

Return: void

set_idt_gate

Signature: void set_idt_gate(void* handler, uint8_t entry_offset, uint8_t type_attr, uint8_t selector)
Description: Sets the interrupt gate for the given interrupt.

Parameters:
  • void* handler
  • uint8_t entry_offset
  • uint8_t type_attr
  • uint8_t selector

Return: void

prepare_interrupts

Signature: void prepare_interrupts()
Description: Sets up the IDT and the interrupt handlers. Also remaps the programable interrupt controller.

Return: void

setup_globals

Signature: void setup_globals(stivale2_struct* bootinfo)
Description: Setup global variables. This function initializes the FontRenderer, MouseRenderer, Renderer2D, DriverManager, SerialDriver, DiskManager, NicManager and Shell.

Parameters:
  • stivale2_struct* bootinfo

Return: void

prepare_acpi

Signature: void prepare_acpi(stivale2_struct* bootinfo)
Description: This function initializes the hpet if available. Also parses the matd table and sets up pci devices either using mcfg or the legacy method.

Parameters:
  • stivale2_struct* bootinfo

Return: void

init_kernel

Signature: KernelInfo init_kernel(stivale2_struct* bootinfo)
Description: Initializes the core kernel features. like the memory manager's, the timer, the framebuffer, the renderer's, the kernel symbol resolver and interrupts.

Parameters:
  • stivale2_struct* bootinfo

Return: KernelInfo
