handle_signal

Signature: bool handle_signal(int signum)
Description: Handels a signal. Calls the signal handler witch the current task has registered. Returns false if there is no handler registered.

Parameters:
  • int signum

Return: bool

register_signal_handler

Signature: void register_signal_handler(int signum, uint64_t handler)
Description: Registers a signal handler for a signal for the current task.

Parameters:
  • int signum
  • uint64_t handler

Return: void
